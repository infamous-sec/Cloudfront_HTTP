# Code of Conduct

All contributors, maintainers, and users of this project are expected to abide by the following Code of Conduct:

## 💬 Be Respectful
We welcome all individuals regardless of background. Discrimination or harassment is not tolerated.

## 🤝 Collaboration First
Critique code, not people. Assume good intentions. We’re all here to improve the project.

## 🛠️ Keep it Constructive
All feedback should be aimed at moving the project forward. Trolls will be ignored and blocked.

## 📬 Contact
If you have issues, email support@shadowlane.cymru or use our [security policy](./SECURITY.md).
