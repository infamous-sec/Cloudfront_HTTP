# Contributing to Cloudfront_HTTP

Thanks for your interest in contributing to ShadowLane SC's code!

## How to Contribute

- Fork the repo
- Clone your fork
- Create a feature branch (`git checkout -b feature-name`)
- Commit with clear messages
- Submit a pull request (PR) to `main`

## Code Guidelines

- Use consistent formatting (2-space indentation)
- Keep `scripts/` secure and out of Git
- Run `npm run build` locally before PR

## Reporting Issues

Please open GitHub issues for bugs, security findings, or suggestions.
