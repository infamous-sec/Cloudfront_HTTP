# ShadowLane SC Website

This is the main website code for ShadowLane SC UK.

## 📦 Folder Contents

- `index.html` – Homepage (replace with production content)
- `contact.php` – Simple contact form (update recipient before production)
- `404.html`, `500.html`, `503.html` – Error/maintenance pages
- `assets/` – Static assets (logo, CSS, JS)
- `.htaccess` – Basic security, HTTPS, custom error pages
- `robots.txt`, `sitemap.xml` – SEO/meta
- `.env.example` – Example environment variables
- `.well-known/security.txt` – Security contact (per standard)
- `humans.txt` – About the team
- `browserconfig.xml`, `site.webmanifest` – Icons & manifest

## 🚀 Deploy via CI/CD

This folder is deployed to cPanel hosting via SFTP using the GitHub Actions workflow in the root repo.
